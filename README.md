# Skeleton for android Single-Activity application
 
 Stack:
  * Kotlin
  * Clean Architecture
  * MVP(Moxy) and MVVM
  * Cicerone 
  * Toothpick
  * Single-Activity with Flows and UI-Scopes (Toothpick) 
  * RxJava and Kotlin Coroutines
  * Material components
